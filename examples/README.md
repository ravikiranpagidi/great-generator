# Examples

Start with `01-generate-from-schema` when you already have a table definition. The examples are ordered from local Pandas usage to Spark and platform writes.

```text
01-generate-from-schema/   Supported schema inputs
02-write-generated-data/  DataFrame writers, Snowflake, Azure SQL, and destinations
03-data-engineering-use-cases/  Lower-environment and pipeline examples
04-custom-rules/           Business-rule overrides
05-generate-from-domain/   Ready-made demo datasets
planned/                   Clearly non-runnable future input designs
```

The older root-level demos remain available for backward-compatible links.
